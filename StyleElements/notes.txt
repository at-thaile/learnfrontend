1. Specify đơn vị đo (measurments) và color:
Relative units: percent,em 
Absolute units: pixed, pointer
Khi ta sử dụng relavative units thì measurments thay đổi khi user thay đổi 
font-size của trình duyệt(browser).
Tuy nhiên, khi ta sử dụng pixed thì nó cũng thay đổi theo 
screen resolution do đó ta cũng thấy thay đổi theo
Khi ta áp dụng font-size 100% cho body thì  font-size thiết lập
100% của font-size browser , thông thường là 16px
1em bằng với current font size
1px bằng với 1 single dot của monitor, số lượng dấu chấm trong mỗi inch
biểu diễn độ phân giải của monitor
pt: pointer: một pointer  = 1/72 của một inch
1em  bằng với font-size của font-size thể cha
1rem: bằng với font-size của cả body
Ví dụ: 
  body{
    font-size: 100%;
  }

  <div class="font" style="font-size: 50%;">
    <p style="font-size: 1.5em">Xin chao toi là le quang thai</p>
    <p style="font-size: 1.5rem">Xin chao toi là le quang thai</p>
  </div>
  Ỏ đây text trong p thứ nhấtsẽ có kích thước = 1.5*0.5*16px = 12px vì thẻ cha
  của nó có font-size = 50% của body
  Còn thẻ p thứ 2 có font-size dựa vào font-size của body = 1.5.16px = 24px;

Lưu ý về độ ưu tiên trong CSS

-inline style > internal style

-inline style > external style

-external style = internal style (cái nào viết trước thì ghi đè lên cái viết sau

<head>
  <style> 
  body { 
   background-color: black; 
  }
  </style> 
  <title>tes test..</title>
   <link rel="stylesheet" href="styles.css"> 
</head>

and in styles.css:

body{ background-color:blue; }

since internal style with black background color came before importing external style from
 styles.css in which background color set as blue, 
 then the final result will be blue background, and vice versa...

the only reason for universal selector to be external style is that it's written in an external css sheet,
 but there's a big difference between ordinary external style selector and universal selector,
this difference is that universal selector can be override by internal style, no matter what's the order of them,
 the internal style will always win and have bigger strength and apply it's changes,
  unlike the ordinary external style selector which have the same strength of internal style 
  and the one who' written last it's styles will be applied (writing order's taken into consideration)


Tóm tắt một cái là universal sẽ luôn luôn bị internal style selector ghi đè lên nó bất kể nó 
đặt trước hay sau
Hãy xem phần selector.html
selector trong CSS : dùng để lựa chọn element tác động

universal selector : * : áp dụng tới tất cả HTML elements
type selector: chỉ gọi tên của element : p,div, a, main, h1,...
id selector : #content lựa chọn các element có id duy nhất
class selector : .right lựa chọn các element có cùng class
